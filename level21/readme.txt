packegeName     : package name of this method
method name     : method's name
method type     : public or friendly or private or protected ?
isStatic        : 0 or 1, static method ?
isFinal         : 0 or 1, final method? 
isAbstract      : 0 or 1, abstract method?
isConstructer   : 0 or 1, constructer method?
isNative        : 0 or 1, native method?
isSynchronized  : 0 or 1, synchronized method?
isGeneric       : 0 or 1, generic method?
hasUnknowArgs   : 0 or 1, has "..." in args?
return value    : method return value type 
generic type    : generic type this method used
args list       : args list
throws          : exception list
